## 创建站点

{% highlight bash %}

gem install jekyll
~ $ jekyll new blog
~ $ cd blog
~/blog $ jekyll serve -w

{% endhighlight %}

`jekyll serve -w` 在本地持续查看最新保存以后的文章内容

`jekyll help server` 查看server命令选项，有很多
